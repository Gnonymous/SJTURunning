# SJTU 校园跑步工具🏃

## 概览

面向 SJTU 校园跑步场景的桌面 GUI 工具：一键登录、生成轨迹并批量上传；参数可在界面或 `config.json` 中配置，适合快速完成多日记录上传。

## 功能特点

- 自动 Jaccount 登录（含验证码识别）
- 自动生成 GPS 跑步轨迹并批量上传
- 支持固定参数或随机距离/配速/时间
- 支持指定日期列表或按天数生成
- 简洁直观的图形界面

## 运行环境

- Python 3.8+
- macOS / Windows / Linux

## 安装与运行

```bash
pip install PySide6 requests tenacity
python qtui.py
```

## 使用步骤

1. 运行 `python qtui.py` 启动图形界面。
2. 输入 Jaccount 用户名与密码。
3. 在界面中调整参数并点击“保存配置”（会写入 `config.json`）。
4. 点击“开始上传”，等待任务完成。

> 提示：账号密码仅用于本次登录，不会写入 `config.json`。

## 配置说明（config.json）

| 参数 | 说明 |
| --- | --- |
| `指定日期模式` | `true` 使用指定日期 / `false` 按天数往前推 |
| `指定日期列表` | 形如 `YYYY-MM-DD` 的日期列表 |
| `跑步天数` | 指定日期模式为 `false` 时生效 |
| `参数随机` | `true` 随机距离/配速 / `false` 固定值 |
| `每日距离_米` | 固定距离（参数随机为 `false` 时生效） |
| `配速_分钟每公里` | 固定配速（参数随机为 `false` 时生效） |
| `距离最小_米` / `距离最大_米` | 随机距离范围 |
| `配速最小_分钟每公里` / `配速最大_分钟每公里` | 随机配速范围 |
| `GPS采样间隔_秒` | 轨迹采样间隔（秒） |
| `跑步时间随机` | `true` 随机时间 / `false` 固定时间 |
| `固定跑步时间_时` / `固定跑步时间_分` | 固定时间（随机为 `false` 时生效） |
| `随机时间范围_开始时` / `随机时间范围_结束时` | 随机时间范围 |
| `起点纬度` / `起点经度` | 轨迹起点 |
| `终点纬度` / `终点经度` | 轨迹终点 |

## 安全与隐私说明

- 登录仅用于获取会话与授权信息，不会保存账号密码。
- 验证码会发送到 `https://geek.sjtu.edu.cn/captcha-solver/` 进行识别。
- 如需更高隐私性，请自行替换验证码识别方式。

## 常见问题

- 登录失败：请检查账号密码、验证码识别服务是否可用。
- 上传失败：可能是网络波动或服务端限制，稍后重试。
- 日期设置：避免提交未来日期或超出规则范围。
